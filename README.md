# LostFound
LOST & FOUND HUB
================

A modern web platform to report, track, and recover lost items using location technology.

FEATURES
--------
• Report lost or found items with detailed forms
• Interactive map with location markers
• Smart search by category, location, keywords
• QR code scanning simulation
• Photo upload ready
• Real-time notifications
• Fully responsive design

TECH STACK
----------
• HTML5, CSS3, JavaScript
• Leaflet.js for maps
• Font Awesome icons
• OpenStreetMap tiles

PROJECT FILES
-------------
index.html    - Main application
css/style.css - All styling
js/script.js  - All functionality

HOW TO DEMO
-----------
1. Show homepage and responsive design
2. Click "Report Found Item" - opens form
3. Click "Report Lost Item" - opens form
4. Use search feature - enter keywords
5. Explore interactive map - click markers
6. Test on mobile - resize browser

ALL BUTTONS WORK:
• Report Found/Lost Items - Opens forms
• Search - Opens search modal
• Map Filters - Change displayed items
• QR Scan - Shows scanner simulation
• Form Submit - Validates and confirms
